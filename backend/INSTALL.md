# تعليمات التثبيت والتشغيل

## تحديث مهم ⚠️

تم تحديث الـ backend لاستخدام `@distube/ytdl-core` بدلاً من `ytdl-core` القديم لحل مشاكل التحميل من YouTube.

## خطوات التثبيت

1. **تثبيت الحزم:**
```bash
cd backend
npm install
```

2. **تشغيل الـ Backend:**
```bash
npm start
```

أو للتطوير مع إعادة التشغيل التلقائي:
```bash
npm run dev
```

## ملاحظات

- تأكد أن الـ Backend يعمل على `http://localhost:3000`
- إذا واجهت مشاكل في التحميل، تأكد من:
  - أن الفيديو غير محمي بحقوق خاصة
  - أن الرابط صحيح
  - أن الإنترنت يعمل بشكل جيد

## استكشاف الأخطاء

إذا ظهرت أخطاء:
1. تحقق من console الـ backend لرؤية التفاصيل
2. تأكد من تثبيت جميع الحزم: `npm install`
3. تأكد من أن FFmpeg مثبت (للتحويل الصوتي)
